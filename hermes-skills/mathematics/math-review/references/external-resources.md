# External Math Resources (collected 2026-07-07)

## Analytic Number Theory

- **Noam Elkies, Harvard Math 259: Introduction to Analytic Number Theory** (Spring 2002-3)
  `https://people.math.harvard.edu/~elkies/M259.02/`
  ~30 lecture note PDFs covering: elementary methods, Dirichlet characters, Riemann zeta function, functional equation, prime number theorem, L-functions, sieve methods, exponential sums, Weil bounds.

## Measure Theory / Real Analysis

- **Jim Belk, Cornell: Measure Theory**
  `https://e.math.cornell.edu/people/belk/measuretheory/`
  19 lectures (12 PDFs available), 11 homework assignments, takehome midterm + final.
