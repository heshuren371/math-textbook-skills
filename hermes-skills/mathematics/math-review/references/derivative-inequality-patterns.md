# 高考导数压轴题——函数构造与不等式证明模式

## 来源

Session 2026-07-07：多道高考导数题（含参不等式恒成立、极值点偏移、隐零点代换）。

---

## 模式 1：含参不等式恒成立

### 标准流程

```
1. 定义 g(x) = f(x,a) ≥ 0
2. 检查边界点：g(0)=0 或 g(1)=0
3. 用边界处的导数获得必要条件：
   - g'(0) ≥ 0  →  a 的粗略下界
   - 若 g'(0)=0，检查 g''(0) ≥ 0
4. 证明充分性：
   - 参变分离 a ≥ g(x) 或 a ≤ g(x)
   - 求 g(x) 的最值
   - 或直接用单调性证 a 在范围内时不等式成立
```

### 案例

- **2020·新课标 I**：\((x-1)e^x \ge mx^2-1\) → \(g'(x)=x(e^x-2m)\) → \(m\le\frac12\)
- **2020秋·沧州期中**：\(\ln x - mx^2 + (1-2m)x + 1 \le 0\) → 最大值在 \(x_0=1/(2m)\) → 最小整数 \(m=1\)
- **2020年山东**：\(ae^{x-1}-\ln x+\ln a\ge1\) → 边界条件 \(a\ge1\)，用隐零点代换证充分

## 模式 2：极值点偏移

### 标准形式

\(f\) 在 \(x_0\) 取极值，\(f(x_1)=f(x_2)\) 且 \(x_1<x_0<x_2\)。

### 常用构造

| 目标 | 构造 |
|------|------|
| \(x_1+x_2>2x_0\) | \(\varphi(x)=f(2x_0-x)-f(x)\)，证 \(\varphi(x)>0\) 在 \(x<x_0\) |
| \(x_1x_2<x_0^2\) | 换元 \(t=\ln x\)，转化为 \(t_1+t_2<2\ln x_0\) |

### 案例

\(f(x)=xe^{-x}\)：\(f(x_1)=f(x_2)\) ⇒ \(x_1+x_2>2\)。  
\(\varphi(x)=f(2-x)-f(x)\)，\(\varphi'(x)=(1-x)(e^{x-2}-e^{-x})<0\) 对 \(x<1\)。  
\(\varphi(1)=0\) ⇒ \(\varphi(x)>0\) ⇒ \(2-x_1<x_2\)。

\(f(x)=x\ln x\)：与 \(y=m\) 交两点 ⇒ \(x_1x_2<1/e^2\)。  
\(t=\ln x\)，\(g(t)=te^t\)，需证 \(t_1+t_2<-2\)。  
\(\varphi(t)=g(-2-t)-g(t)\)，\(\varphi'(t)=(1+t)(e^{-2-t}-e^t)<0\) 对 \(t<-1\)。

## 模式 3：隐零点代换

\(f'(x)=0\) 无法显式求解时：设 \(x_0\) 满足 \(f'(x_0)=0\)，用此关系消去 \(f(x_0)\) 中的指数/对数项，得到 \(f(x_0)\) 仅关于 \(x_0\) 的表达式，再证其 ≥ 0。

### 案例

**2020山东**：\(f'(x)=ae^{x-1}-1/x=0\) ⇒ \(ae^{x_0-1}=1/x_0\)，\(\ln a=1-x_0-\ln x_0\)。  
\(f(x_0)=1/x_0-x_0+1-2\ln x_0 = h(x_0)+1\)，\(h(x)=1/x-x-2\ln x\) 严格减，\(h(1)=0\)，故 \(x_0\le1\) 时 \(f(x_0)\ge1\)。

## 模式 4：三角不等式 \(\sin x\pm\cos x\) 的简化

\[
\frac{\sin x-\cos x}{\sin x+\cos x} = \frac{\tan x-1}{\tan x+1} = \tan\!\Bigl(x-\frac\pi4\Bigr)
\]

用于将 \(f(x)=(\sin x-\cos x)-(x+a)(\sin x+\cos x)\ge0\) 转化为 \(a\le\tan(x-\pi/4)-x\) 形式的参变分离。
