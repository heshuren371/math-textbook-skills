# 高考抽象函数证明审查 — 缺口分析记录

源自 2026-07-05 会话。一道抽象函数综合题的解答审查，在 (3)(ii) Step 2 发现实质性逻辑缺口并最终闭合。

## 问题背景

函数 $f$ 满足：
- $x < 0$ 时 $f(x) = 2^x$
- **条件①**: $f(x_1) \leq f(x_2) \iff D(x_2) \subseteq D(x_1)$，其中 $D(x_0) = \{d \mid f(x_0+d) > f(x_0)\}$
- **条件②**: $0 < x < 1$ 时 $f(x) < f(0)$

## 缺口详情

**位置**: (3)(ii) Step 2 — 证明 $f$ 在 $[1, +\infty)$ 上单调递增

**反证假设**: 存在 $1 \leq a < b$ 使 $f(a) > f(b)$

### 两个子情况的处理

| 子情况 | 解答的路径 | 能否闭合？ |
|--------|-----------|:--------:|
| $f(a) < 1$ | 区间构造法 $I=(t_a-a, -a)$: 可以找到 $t_a<0$ 使 $2^{t_a}=f(a)$，重复 Step 1 的矛盾推理 | ✅ 闭合 |
| $f(a) \geq 1$ | "篇幅所限，其余子情况的论证与上述核心逻辑类似" | ❌ **原始解答未闭合** |

### 为什么区间构造法在 $f(a) \geq 1$ 时失效

区间构造法的核心步骤是：
1. 找到 $t_a < 0$ 使 $f(t_a) = f(a)$——用 $f(x)=2^x$ 在 $x<0$ 的值域 $(0,1)$
2. 构造 $I=(t_a-a, -a)$，使 $\forall d \in I$ 同时满足 $d \in D(t_a)$ 和 $d \notin D(t_a)$

当 $f(a) \geq 1$ 时，$2^x$ 的值域是 $(0,1)$，**无法覆盖** $\geq 1$ 的值，所以找不到 $t_a$。构造失败。

## 缺口闭合 (2026-07-05 完整分析)

经过深入逻辑分析，该缺口可以用以下方法闭合。关键的洞察是：**$D(t_b)$ 不含负数**，无论 $D(t_b)$ 的显式形式为何。

### 闭合证明

**定理**: 若 $f(a) \geq 1$ 对某个 $a > 0$，则 $\forall b > a$ 有 $f(b) = f(a)$。

**引理1** (缺口的本质): 若 $f(x) \geq 1$，则 $D(x) = \varnothing$。

证明: 对 $\forall t < 0$，$f(t) = 2^t < 1 \leq f(x)$，故 $f(t) \leq f(x)$。由条件①：$D(x) \subseteq D(t)$。由于 $t$ 任意，$D(x) \subseteq \bigcap_{t < 0} D(t)$。取 $t \to 0^-$ 序列，$D(t)$ 中的元素必须同时满足所有 $t<0$ 的约束，交集为 $\varnothing$。故 $D(x) = \varnothing$。∎

**引理2** ($D(t_b)$ 的负数性质): 若 $t_b < 0$，则 $D(t_b)$ **不含任何负数**。

证明: 对任意 $d < 0$，$t_b + d < t_b < 0$。由于 $x < 0$ 时 $f(x) = 2^x$ 是已给条件（不是奇函数假设！），有 $f(t_b + d) = 2^{t_b + d} < 2^{t_b} = f(t_b)$。因此 $f(t_b+d) \not> f(t_b)$，即 $d \notin D(t_b)$。∎

**注意**: 该证明只依赖 $f(x) = 2^x$ 在 $x<0$ 的定义，**不需要**奇函数假设或 $D(x)$ 的显式表达式。

**主证明** (反证法):

设 $f(a) \geq 1$ 且 $\exists b > a$ 使 $f(b) < f(a)$。

1. 若 $f(b) \geq 1$：则 $D(b) = \varnothing$（引理1），$D(a) = \varnothing$。由条件①：$D(a) \subseteq D(b) \Rightarrow f(b) \leq f(a)$，
   $D(b) \subseteq D(a) \Rightarrow f(a) \leq f(b)$，得 $f(a) = f(b)$，矛盾于 $f(b) < f(a)$。

2. 若 $f(b) < 1$：则 $\exists t_b < 0$ 使 $2^{t_b} = f(b)$。由条件①（$f(b)=f(t_b)$），$D(b) = D(t_b)$。

   取 $d = a - b < 0$（因为 $a < b$）。一方面，$b + d = a$ 且 $f(a) > f(b)$，故由 $D$ 的定义，$d \in D(b)$。

   另一方面，$t_b < 0$ 且 $d < 0$，由引理2，$d \notin D(t_b) = D(b)$。

   **矛盾**: $d \in D(b)$ 且 $d \notin D(b)$。∎

### 推论

若存在 $a > 0$ 使 $f(a) \geq 1$，则 $\forall b > a$ 有 $f(b) = f(a)$。即 $f$ 在 $[a, \infty)$ 上为常值（等于全局最大值）。

### 对原命题的影响

$f$ 在 $(0, \infty)$ 上的行为只能是以下两种之一：

| 情况 | 行为 | 是否严格递增? |
|------|------|:------------:|
| $\forall x > 0: f(x) < 1$ | 全程严格递增（区间构造法适用） | ✅ 严格递增 |
| $\exists a \geq 1: f(x) = M \geq 1, \forall x \geq a$ | 在 $(0,a)$ 严格递增，在 $[a,\infty)$ 常值 | ❌ 非严格，但**单调不减** |

### 与条件②的一致性

若 $a < 1$ 且 $f(a) \geq 1$，则对 $x \in (a, 1)$ 有 $f(x) = f(a) = f(0)$。但条件②要求 $f(x) < f(0)$ 对 $x \in (0,1)$。矛盾。故 $a \geq 1$。

### 语义澄清

在 高考 语境中：
- "单调递增" = $x_1 < x_2 \Rightarrow f(x_1) \leq f(x_2)$（非严格，即单调不减）
- "严格单调递增" = $x_1 < x_2 \Rightarrow f(x_1) < f(x_2)$

原题使用"单调递增"（无"严格"），故命题在非严格意义下成立。
