# 高考含参不等式恒成立问题

## 典型结构

\(f(x) \ge 0\)（或 \(f(x) \le 0\)）\(\forall x \in D\)，其中 \(f(x)\) 含参数，求参数的取值范围。

## 标准工作流

### Step 1 — 参变分离

将含参不等式化为 \(a \ge g(x)\) 或 \(a \le g(x)\) 的形式。

例1：\(e^x+ax^2-x-\frac12x^3-1\ge0 \Longrightarrow a \ge \dfrac{x+\frac12x^3+1-e^x}{x^2}\)

例2：\(\ln x - mx^2 + (1-2m)x + 1 \le 0\) 对 \(x>0\) 恒成立 → 求使 \(\max f(x)\le0\) 的整数 \(m\)

**检查**：分离时分母在定义域内的符号是否不变。

### Step 2 — Taylor 展开确定必要条件

在 \(x=0\) 处展开 \(h(x)=f(x,a)\)，最低阶非零项的系数给出参数的必要范围。

例：\(h(x)=e^x+ax^2-x-\frac12x^3-1 = (a+\frac12)x^2 - \frac13x^3 + \cdots\)  
\(x\to0^+\) 时由 \((a+\frac12)x^2\ge0\) 得 \(a>-\frac12\) 为必要条件。

### Step 3 — 求 \(g(x)\) 的极值

对分离后的函数 \(g(x)\) 求导，将 \(g'(x)\) 分子因式分解为 \((x-x_0)\varphi(x)\)，其中 \(\varphi(x)\) 在定义域内定号。

例：\(g'(x)=\dfrac{(x-2)(x^2+2x-2e^x+2)}{2x^3}\)  
令 \(\varphi(x)=x^2+2x-2e^x+2\)，\(\varphi(0)=0\)，\(\varphi'(x)=2x+2-2e^x<0\) 对 \(x>0\) 恒成立 → \(\varphi(x)<0\) 定负 → 唯一驻点 \(x=2\) 且为最大值点。

### Step 4 — 代入求阈值

\(a_{\text{boundary}} = g(x_{\text{critical}})\)

例：\(g(2)=\dfrac{7-e^2}{4}\)

### Step 5 — 整数优化（参数为整数时）

找到实数阈值后，取比阈值大的最小整数。

例：\(f_{\max}=-\ln(2m)+\frac1{4m}\le0\) → \(\ln(2m)\ge\frac1{4m}\)  
\(m=1\)：\(\ln2\approx0.693>0.25\) ✓ → \(m_{\min}=1\)

### Step 6 — 充分性验证

取临界参数值，用 SymPy 数值验证。

```python
a_val = (7 - sp.exp(2))/4
h = sp.exp(x) + a_val*x**2 - x - x**3/2 - 1
# 扫描 x≥0 确认 h(x) ≥ 0
```

## Pitfalls

| 陷阱 | 后果 | 修复 |
|------|------|------|
| 参变分离忽略分母变号 | 符号反向 | 先确认分母定号 |
| 只找临界点不检查边界 | 遗漏端点极值 | 同时检查 \(x\to0^+\) 和 \(x\to\infty\) |
| Taylor 必要条件当作充分条件 | 阈值不准确 | 精确边界依赖 \(g(x)\) 分析 |
