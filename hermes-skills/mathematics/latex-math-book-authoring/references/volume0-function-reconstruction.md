# 第0编：函数基础重建（8章，127页）

## 定位

从"零函数基础"到"具备数学分析全部预备知识"。读者起点为代数运算熟练但函数概念几乎空白（初三毕业后未学数学）。目标是为数列极限（ε-N）、函数极限（ε-δ）、导数、积分、级数等严格分析内容铺平道路。

## 8章路线

```
第1章 函数的概念       定义域/值域/对应法则/分段函数/函数与方程
第2章 函数的图像       描点法/六种基本函数/平移对称翻折/垂线检验法
第3章 幂函数与多项式    正整数幂/负整数幂/分数幂/奇偶性/多项式根/重根
第4章 指数函数          底>1增长/0<底<1衰减/对称关系/自然指数e^x/指数vs幂函数/Sofmax
第5章 对数函数          定义/运算法则/换底公式/对数函数图像/指数与对数互逆/压缩效果/交叉熵
第6章 三角函数(上)      弧度制/单位圆/sin与cos图像/周期振幅/相位关系/一般形式

```

## 每章模板结构

每章按以下7段顺序书写，不可调换：

1. **从上一章来**（3-8行过渡桥梁，连接上一章终点与本章起点）
2. **应用引导框**（`\begin{applicationbox}` — 三问：学之前做不到/学之后能做到/日常例子）
3. **核心内容**（问题引入 → 直觉引入 → 定义框 → 8-12个例题，第一个完全展开不跳步）
4. **符号卡片**（`\begin{symbolcard}` — 符号/LaTeX写法/读法/含义）
5. **代码验证**（Python lstlisting + 运行结果，验证关键公式/定理）
6. **本章小结**（`\begin{progressbox}` — 核心知识点列表）
7. **通往下一章**（3-8行过渡桥梁）
8. **习题**（`\section{习题}` — 30道，9级难度，无竞赛）

## 图的质量规则（用户反馈驱动）

用户反馈"之前的教材图像太少"、"U型函数的顶点是错的"、"坐标系有两个O点"：
- 每章至少5幅矢量图（第2章8张、第3章6张、第4章5张、第5章4张、第6章5张、第7章4张、第8章3张）
- 每节至少1幅图，抽象概念必须配图
- 每个顶点用大红点（`s=100+`, `zorder=5`, `edgecolors='black'`）标注，箭头指向精确
- 原点处最多标一个东西（刻度`0`足够），不额外标`O`或`(0,0)`
- matplotlib标题中不用emoji/Unicode特殊字符（`✓`→`[是]`，`✗`→`[不是]`）
- 几何图设`set_aspect('equal')`
- 图必须在正文中被引用（`\includegraphics` + 文字说明"如图X所示"）
