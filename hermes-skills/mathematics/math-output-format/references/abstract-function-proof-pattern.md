# 抽象函数证明方法库

## 问题特征

条件形如：

```
条件①: f(x₁) ≤ f(x₂) ⇔ D(x₂) ⊆ D(x₁)   其中 D(x) = {d | f(x+d) > f(x)}
条件②: 在某个区间上 f(x) 满足特定不等式
```

外加部分区域上函数有显式表达式（如 \(f(x) = 2^x\) 当 \(x<0\)）。

证明目标通常是：单调性、有界性、最值。

---

## 核心套路

### 1. 结构等价传递

若 \(f(x) = f(t)\)，则由条件①：

```
f(x) ≤ f(t) ⇒ D(t) ⊆ D(x)
f(t) ≤ f(x) ⇒ D(x) ⊆ D(t)
∴ D(x) = D(t)
```

用「已知形式」的 \(D(t)\) 去确定「未知形式」的 \(D(x)\)。

### 2. 区间构造法 (Interval Construction)

当 \(f(a) < 1\) 时，存在 \(t_a < 0\) 使 \(f(t_a)=2^{t_a}=f(a)\)，且 \(D(a)=D(t_a)\)。

对任意 \(b > a\) 假设 \(f(b) \leq f(a)\)，则 \(D(a) \subseteq D(b)\)。

构造区间 \(I = (t_a - a,\; -a)\)：

| 条件 | 验证 |
|------|------|
| \(I \neq \varnothing\) | \(t_a - a < -a \iff t_a < 0\) ✅ |
| \(d < 0\) | \(d < -a < 0\) ✅ |
| \(a+d < 0\) | \(d < -a\) ✅ |
| \(a+d > t_a\) | \(d > t_a - a\) ✅ |
| \(t_a+d < 0\) | \(t_a + d < t_a < 0\) ✅ |

对任意 \(d \in I\)：

- \(a+d < 0\) 且 \(a+d > t_a \Rightarrow 2^{a+d} > 2^{t_a} = f(a) \Rightarrow d \in D(a) = D(t_a)\)
- \(t_a+d < t_a < 0\) 且 \(d < 0 \Rightarrow 2^{t_a+d} < 2^{t_a} \Rightarrow d \notin D(t_a)\)

→ 矛盾 ✅。故 \(f(b) > f(a)\)。

**适用范围**：任意 \(a > 0\) 且 \(f(a) < 1\)，不要求 \(a < 1\)。

### 3. D(a) = ∅ 引理

若 \(f(a) \geq 1\)：

对任意 \(t < 0\)：\(f(t) = 2^t < 1 \leq f(a) \Rightarrow f(t) \leq f(a) \Rightarrow D(a) \subseteq D(t)\)。

而 \(D(t) \supseteq (0, -t]\)（所有正的小偏移 \(d\) 满足 \(t+d<0 \Rightarrow 2^{t+d}>2^t\)）。

取 \(t \to 0^-\)：\((0, -t) \to (0, 0^+)\)，交集为空。故 \(D(a) = \emptyset\)。

**推论**：\(f(a) \geq 1 \Rightarrow f(a)\) 是全局最大值。

### 4. 负数闯入矛盾

这是最隐蔽的缺口来源。当 \(f(b) < 1\) 时：

```
∃ t_b < 0: f(t_b) = 2^{t_b} = f(b) ⇒ D(b) = D(t_b)
```

对任意 \(d < 0\)：\(t_b + d < t_b < 0 \Rightarrow f(t_b+d)=2^{t_b+d} < 2^{t_b} = f(t_b)\)，故 \(d \notin D(t_b)\)。

**因此 \(D(b) = D(t_b)\) 不含任何负数。**

现在假设 \(f(a) \geq 1\) 且 \(f(a) > f(b)\)（即 \(f\) 在 \(a\) 之后下降）：

- \(a - b < 0\)（因为 \(a < b\)）
- \(b + (a-b) = a\)，且 \(f(a) > f(b)\)
- 由定义：\(a - b \in D(b)\)
- 但 \(D(b) = D(t_b)\) 不含负数！

→ 矛盾。故 \(f(b) \geq f(a)\)。

结合 \(f(a)\) 为全局最大值（D(a) = ∅），得 \(f(b) = f(a)\)：

> 若 \(f\) 在某点达到 \(\geq 1\)，则之后恒为常值。

---

## 完整证明链

```
前提: f(x) = 2^x (x<0), 条件①②
  │
  ├─(3)(i) 反证法: f(0) < 1
  │   ⇒ 构造 x₀ ⇒ (0,-x₀) ⊆ D(0) 与条件②矛盾 ⇒ f(0) ≥ 1
  │
  └─(3)(ii) 单调性:
       │
       ├─ 引理: ∀x∈(0,1), f(x) < 1
       │
       ├─ Step 1: f 在 (0,1) 上严格递增
       │   (区间构造法 I = (t_a-a, -a))
       │
       └─ Step 2: f 在 (0,∞) 上单调不减
            │
            ├─ 情况A: f(a) < 1 ⇒ 区间构造法 ⇒ 严格递增
            │
            └─ 情况B: f(a) ≥ 1
                 ├─ Lemma A: D(a) = ∅ ⇒ f(a) 全局最大
                 ├─ Lemma B: f(b) < f(a) ⇒ 负数闯入矛盾
                 └─ ∴ f(b) = f(a) ⇒ 常值（单调不减）
```

**注意**：严格递增在 \(f(a) \geq 1\) 时不可证。正确的结论是单调不减（non-decreasing）。

## 自我检查清单

- [ ] 条件①的「iff」是否同时用了两个方向？
- [ ] 区间 I 非空的验证是否显式写出来了？
- [ ] D(a) = ∅ 的推导是否依赖了奇函数假设（可能已在另一小题中废弃）？
- [ ] 分类讨论是否覆盖了「f(a) ≥ 1」和「f(a) < 1」两个互斥情况？
- [ ] 负数闯入矛盾中：\(a-b < 0\) 和 \(a-b \in D(b)\) 是否同时成立？
- [ ] 结论是「严格递增」还是「单调不减」？命题是否在给定条件下可证？
