# 几何正确性验证指南

教材图的几何关系必须严格正确，任何逻辑错误都会导致学生学到错误概念。本文档为每类图提供可复现的验证方法。

---

## 一、同位角与内错角图

### 原则
截线与两条平行线相交，同位角在两交点处位于**同一方位**，内错角位于**平行线内侧**。

### 验证步骤

1. 确定截线斜率 `m = tan(θ)`
2. 计算交点坐标：解联立方程 `y = m*x` 与 `y = y_parallel`
3. 同位角弧线：从平行线水平方向开始，到截线方向。两交点处起始/终止角度完全相同。

```python
import numpy as np
m = np.tan(np.radians(40))        # 截线斜率
xu = 1.6 / m                       # 上交点 x

# 同位角：从 0°（右）到 40°（截线方向）
arc = Arc((xu, 1.6), 1.0, 1.0, theta1=0, theta2=40, ...)

# 内错角：在平行线内侧，与同位角互补
# 上交点：从 220° 到 360°（即 -140° 到 0°，内侧角）
# 下交点：从 40° 到 180°（内侧角）
```

### 常见错误
- `❌ theta1=45, theta2=90`— 截线不一定是 45°，且 45→90 不是从平行线方向开始
- `❌` 两交点处同位角弧线位于不同方位

## 二、面积法证明勾股定理

### 原则
大正方形边长 a+b，四角放 4 个**全等直角三角形**，中间留下边长为 c 的正方形。

### 验证步骤

1. 确定直角三角形的两条直角边分别为 a 和 b
2. 确保四个三角形的斜边都是 `c = sqrt(a² + b²)`
3. 验证中间四边形的四条边都等于 c

```python
a, b = 4, 3
s = a + b

# 四个三角形，每个的直角边分别为 a 和 b
triangles = [
    [(0, 0), (b, 0), (0, a)],          # 左下：直角在(0,0)
    [(s, 0), (s-a, 0), (s, b)],        # 右下：直角在(s,0)
    [(s, s), (s-b, s), (s, s-a)],      # 右上：直角在(s,s)
    [(0, s), (a, s), (0, s-b)],        # 左上：直角在(0,s)
]

# 验证：每个三角形的三条边
for tri in triangles:
    x1, x2, x3 = [p[0] for p in tri]
    y1, y2, y3 = [p[1] for p in tri]
    d12 = ((x1-x2)**2 + (y1-y2)**2)**0.5
    d23 = ((x2-x3)**2 + (y2-y3)**2)**0.5
    d31 = ((x3-x1)**2 + (y3-y1)**2)**0.5
    edges = sorted([d12, d23, d31])
    assert abs(edges[0] - a) < 1e-9, f"短边 {edges[0]} ≠ a={a}"
    assert abs(edges[1] - b) < 1e-9, f"中边 {edges[1]} ≠ b={b}"
    assert abs(edges[2] - (a**2+b**2)**0.5) < 1e-9, f"斜边 {edges[2]} ≠ c"

# 中间正方形
center = [(b, 0), (s, b), (s-b, s), (0, a)]
for i in range(4):
    p1, p2 = center[i], center[(i+1)%4]
    d = ((p1[0]-p2[0])**2 + (p1[1]-p2[1])**2)**0.5
    assert abs(d - (a**2+b**2)**0.5) < 1e-9, f"中间边 {d} ≠ c"
```

### 常见错误
- `❌` 右下三角形的两条直角边都是 b（应该一个是 a，一个是 b）
- `❌` 左上三角形的两条直角边都是 a（同理应该各为 a 和 b）
- `❌` 中间四边形不是正方形（验证四条边是否都等于 c）

## 三、3D 立体图（圆柱/圆锥）

### 原则
椭圆描述底面/顶面，侧面线必须**精确连接**椭圆长轴端点。

### 验证步骤

1. 椭圆方程：`(x-cx)²/r² + (y-cy)²/(h_ellipse/2)² = 1`
2. 侧面线起点必须在椭圆边界上
3. 圆锥母线终点必须精确为顶点 `(cx, y_apex)`

```python
r = 1.5           # 椭圆半长轴
h_e = 0.75        # 椭圆短轴（高度）
y_bottom = 0.5
y_top = y_bottom + 3.5

# 圆柱侧面线：连接上下椭圆长轴端点
ax.plot([-r, -r], [y_bottom, y_top], '-', color=C_MAIN, lw=1.8)
ax.plot([r, r], [y_bottom, y_top], '-', color=C_MAIN, lw=1.8)

# 圆锥母线：从椭圆端点连到顶点
ax.plot([-r, 0], [y_bottom, y_top], '-', color=C_MAIN, lw=1.8)
ax.plot([r, 0], [y_bottom, y_top], '-', color=C_MAIN, lw=1.8)

# 验证：起点在椭圆上
# 对于底面椭圆 (0, y_bottom)，端点 (-r, y_bottom) 满足：
# (-r - 0)² / r² + (y_bottom - y_bottom)² / (h_e/2)² = r²/r² + 0 = 1 ✓
```

### 常见错误
- `❌` 母线起点 y 坐标不在椭圆边界上（如 `y=0` 而椭圆在 `y=0.5`）
- `❌` 圆锥母线终点 x 坐标偏离顶点（如 `(0.5, y_apex)` 而非 `(0, y_apex)`）
- `❌` 被遮挡的后半椭圆画成实线（应该用 `linestyle='--'`）

## 四、三角形标签与顶点对应

### 原则
文字标签描述的顶点必须与实际绘制的坐标完全一致。

### 验证方法

```python
# 定义顶点
A, B, C = (0, 0), (3, 0), (1.5, 2)

# 绘制三角形
draw_triangle(ax, [A, B, C])

# 标签必须与定义一致
ax.text(A[0]-0.1, A[1]-0.1, '$A$', ha='right', va='top')
ax.text(B[0]+0.1, B[1]-0.1, '$B$', ha='left', va='top')
ax.text(C[0], C[1]+0.1, '$C$', ha='center', va='bottom')
```

### 常见错误
- `❌` 用 `ax.plot([0,0],[0,2])` 画出折线，但标签写的是三角形顶点
- `❌` 三角形定义的顶点与标签描述的完全不同

## 五、角标记与实际角度一致

### 原则
角标记的形状必须与实际角度大小一致。

### 验证方法

```python
import numpy as np

# 三角形顶点
A, B, C = (0, 0), (3, 0), (1.5, 2)

# 计算角 C 的实际大小
v1 = np.array([A[0]-C[0], A[1]-C[1]])  # C→A
v2 = np.array([B[0]-C[0], B[1]-C[1]])  # C→B
cos_angle = np.dot(v1, v2) / (np.linalg.norm(v1) * np.linalg.norm(v2))
actual_angle = np.degrees(np.arccos(cos_angle))
print(f"角 C 实际大小: {actual_angle:.1f}°")

# 小三角形标记开口必须等于 actual_angle
# 如果画的是固定 53° 小三角形，而实际是 74°，就是错误的
```

### 常见错误
- `❌` 用固定尺寸的小三角形标记所有角，不管实际角度大小
