# 中册图形修复案例（2026-07-12）

本文档记录了一次完整的几何图审查与批量修复过程，作为未来类似任务的参考。

---

## 背景

用户反馈：「所有的矢量图做的其实都不好」

审查范围：中册教材 `figures/` 目录下 27 张 PDF 矢量图
审查结果：**FAIL** — 存在多处几何事实错误

---

## 错误清单（按严重级别）

### ★ 阻塞级—几何事实错误

| 图名 | 错误描述 | 后果 |
|------|----------|------|
| `ch14_angle_relations` | 同位角弧 `theta1=45, theta2=90`（截线实际 40°）；内错角弧与平行线/截线夹角无关 | 学生看到的“同位角”不是同位角 |
| `ch14_parallel` | `ax.plot([[-2.8,0.8],[-2.8,-0.8]], ...)` 语法错误—传入 2×2 数组，画出的不是垂直线 | 平行线“距离处处相等”图示缺失或错误 |
| `ch14_angle_proof_strict` | 标签说 A=(0,0), B=(3,0), C=(1,2.1)，但实际绘制的是折线 (0,0)–(0,2)–(2,0)–(3,0) | 内角和证明的图示对象与文字描述完全脱节 |
| `ch16_area_proof` | 右下三角形两条直角边都是 b=3；左上三角形两条直角边都是 a=4；中间四边形不是正方形 | 面积法证明勾股定理的**核心图示是错的** |
| `ch17_cylinder` | 圆柱侧面连接线 `[-1.75,0.5], [-1.75,3.5]` — y 从 -1.75 跳到 3.5，完全脱离椭圆 | 圆柱体变成了随机四边形 |
| `ch17_cone` | 圆锥母线 `[-1.75,0.5], [0,4]` — 起点 y=0 在底面椭圆下方，终点 x=0.5 偏离顶点 | 圆锥母线不经过顶点 |

### ● 视觉规范问题

1. `bbox_inches='tight'` 导致边距不一致 — 每张图插入 LaTeX 后实际尺寸不可预测
2. 无虚实线区分 — 3D图（长方体、棱柱）中被遮挡的边应画虚线
3. 颜色与线宽不统一 — 同一张图中辅助线、标注线、主体图形没有视觉层次
4. 角标注不精确 — `ch14_tri_angles` 中角C的小三角开口角(≈53°)与实际角度(≈74°)不符
5. 标注文字拥挤 — 多次出现文字紧贴图形或超出边界

---

## 修复方案

### 1. 绘图规范标准化

建立了一套统一参数：

```python
C_MAIN = '#000000'   # 主体线
C_AUX  = '#555555'   # 辅助线
C_BLUE = '#1a5276'   # 蓝色标注
C_RED  = '#922b21'   # 红色标注
C_FILL = '#eaf2f8'   # 填充背景
```

线宽层次：主体 lw=2.0，辅助 lw=1.0，遮挡虚线 linestyle='--'
标注间距：文字与图形边缘 ≥ 0.15 坐标单位

### 2. 固定边距控制

原问题：`bbox_inches='tight'` 会让每张图的最终尺寸不同。

修复：固定 `figsize` + `set_xlim`/`set_ylim` 统一定义画布范围，
保存时仍用 `bbox_inches='tight'` 但配合 `pad_inches=0.08`，
关键是靠 `set_xlim`/`set_ylim` 而非 `tight` 来控制边距。

### 3. 具体图形修复示例

#### ch14_angle_relations — 同位角/内错角

原错误：`theta1=45, theta2=90` 凭感觉填写

修复：截线斜率 `m = tan(40°)`，交点处同位角弧从水平方向开始：
```python
m = np.tan(np.radians(40))
xu, yu = 1.6/m, 1.6   # 上交点
xl, yl = -1.6/m, -1.6  # 下交点

# 同位角：两交点都是 0°→40°
arc = Arc((xu, yu), 1.0, 1.0, theta1=0, theta2=40, ...)
arc = Arc((xl, yl), 1.0, 1.0, theta1=0, theta2=40, ...)

# 内错角：两交点内侧，互补至 180°
arc = Arc((xu, yu), 1.0, 1.0, theta1=220, theta2=360, ...)  # 上交点内侧
arc = Arc((xl, yl), 1.0, 1.0, theta1=40, theta2=180, ...)   # 下交点内侧
```

#### ch14_angle_proof_strict — 三角形顶点对应

原错误：标签与绘制的图形完全脱节

修复：使用单一源头定义顶点，标签与之一一对应：
```python
A, B, C = (0, 0), (4, 0), (1.2, 2.0)
ax.plot([A[0], C[0]], [A[1], C[1]], '-', lw=1.8)
ax.plot([C[0], B[0]], [C[1], B[1]], '-', lw=1.8)
ax.plot([A[0], B[0]], [A[1], B[1]], '-', lw=1.8)

ax.text(A[0]-0.15, A[1]-0.15, '$A$', ...)
ax.text(B[0]+0.15, B[1]-0.15, '$B$', ...)
ax.text(C[0], C[1]+0.12, '$C$', ...)
```

#### ch16_area_proof — 面积法证明

原错误：四个三角形不全等，中间不是正方形

修复：每个三角形的两条直角边必须分别为 a 和 b：
```python
a, b = 4, 3
s = a + b

triangles = [
    [(0, 0), (b, 0), (0, a)],          # 左下：直角在(0,0)
    [(s, 0), (s-a, 0), (s, b)],        # 右下：直角在(s,0)
    [(s, s), (s-b, s), (s, s-a)],      # 右上：直角在(s,s)
    [(0, s), (a, s), (0, s-b)],        # 左上：直角在(0,s)
]
```

#### ch17_cylinder — 圆柱侧面线

原错误：侧面连接线 y 坐标脱离椭圆

修复：连接线必须精确连接椭圆长轴端点：
```python
r = 1.5
y_bottom, y_top = 0.5, 4.0

# 侧面线从椭圆长轴端点 x=±r 开始
ax.plot([-r, -r], [y_bottom, y_top], '-', lw=1.8)
ax.plot([r, r], [y_bottom, y_top], '-', lw=1.8)

# 下椭圆被遮挡的后半部分画虚线
arc = Arc((0, y_bottom), 2*r, 0.75, theta1=180, theta2=360,
          lw=1.2, linestyle='--')
```

#### ch17_cone — 圆锥母线

原错误：母线不经过顶点

修复：母线从椭圆端点精确连到顶点：
```python
r = 1.5
y_base, y_apex = 0.5, 4.0

ax.plot([-r, 0], [y_base, y_apex], '-', lw=1.8)
ax.plot([r, 0], [y_base, y_apex], '-', lw=1.8)
```

## 预防措施

1. **建立统一绘图规范脚本**—所有图使用相同的颜色、线宽、字体、边距参数
2. **几何正确性检查清单**—每张图交付前通过相应类别的验证
3. **标签与坐标一一对应**—通过单一源头定义顶点坐标
4. **角度精确计算**—使用 `np.arctan2()` 而非凭感觉填写弧线参数
5. **连接线坐标验证**—验证起点在椭圆/圆上，终点精确为顶点
