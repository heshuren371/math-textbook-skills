# 严格模式证明模板（ε-N / ε-δ）

当教材以「严格模式」（数学分析/理论导向）运行时，以下关键定理需要 ε-N/ε-δ 证明链。按依赖顺序排列。

---

## 1. 数列极限：ε-N 证明模板

**目标：** 证 \(\lim a_n = L\)

**模板：**
```
对任意 ε > 0，要找 N 使 n > N 时 |a_n - L| < ε。
从 |a_n - L| 出发，放大到关于 n 的简单形式（如 C/n），
令 C/n < ε ⇒ n > C/ε。
取 N = ⌈C/ε⌉（或更大的整数），则 n > N 时 |a_n - L| < ε。
```

**经典例子：**
- \(1/n \to 0\)：\(N = ⌈1/ε⌉\)
- \(n/(2n+1) \to 1/2\)：\(N = ⌈1/(4ε)⌉\)
- \(1/2^n \to 0\)：\(N = ⌈\ln(1/ε)/\ln 2⌉\)

---

## 2. 函数极限：ε-δ 证明模板

**目标：** 证 \(\lim_{x \to a} f(x) = L\)

**模板：**
```
对任意 ε > 0，要找 δ > 0 使 0 < |x-a| < δ 时 |f(x)-L| < ε。
从 |f(x)-L| 出发，用不等式放大到关于 |x-a| 的简单形式（如 C|x-a|），
令 C|x-a| < ε ⇒ |x-a| < ε/C。
取 δ = ε/C，则满足条件。
```

**关键技巧：** 对多项式用因式分解 \(|f(x)-L| = |x-a|·|g(x)|\)，再对 \(|g(x)|\) 在 a 附近取上界。

---

## 3. 洛必达法则的 ε-δ 证明

**条件：** \(f(a)=g(a)=0\)，\(\lim_{x\to a} f'(x)/g'(x) = L\)

**证明框架：**
```
对任意 ε > 0，存在 δ > 0 使 0 < |x-a| < δ 时 |f'(x)/g'(x) - L| < ε。
对 x ∈ (a, a+δ)，由柯西中值定理：
  (f(x)-f(a))/(g(x)-g(a)) = f'(c)/g'(c) 对某 c ∈ (a,x)
故 |f(x)/g(x) - L| = |f'(c)/g'(c) - L| < ε。
```

## 4. 泰勒公式的余项证明

**拉格朗日余项（ε-δ 框架）：**
```
令 F(t) = f(t) + ∑_{k=1}^{n-1} f^{(k)}(t)(x-t)^k/k!，
G(t) = (x-t)^n。
由柯西中值定理，存在 ξ ∈ (a,x) 使
  (F(x)-F(a))/(G(x)-G(a)) = F'(ξ)/G'(ξ)
化简得 R_n(x) = f^{(n)}(ξ)(x-a)^n / n!
```

## 5. 黎曼积分的 ε-δ 可积性证明

**定理：** 连续函数在闭区间上黎曼可积。

**证明框架：**
```
f 在 [a,b] 上连续 ⇒ 一致连续。
对任意 ε > 0，存在 δ > 0 使 |x-y| < δ 时 |f(x)-f(y)| < ε/(b-a)。
取分割 P 使最大宽度 < δ，记 U(P,f) - L(P,f) = ∑(M_i - m_i)Δx_i
  < ε/(b-a)·∑Δx_i = ε
由达布定理，f 黎曼可积。
```

## 6. 微积分基本定理的 ε-δ 证明

**定理：** \(\Phi(x) = \int_a^x f(t)dt\) ⇒ \(\Phi'(x) = f(x)\)

**证明框架：**
```
对固定 x，取 h ≠ 0：
  (Φ(x+h)-Φ(x))/h = (1/h)∫_x^{x+h} f(t)dt = f(ξ)（积分中值）
由 f 连续，对 ε>0 存在 δ>0 使 |t-x|<δ ⇒ |f(t)-f(x)|<ε。
取 |h|<δ ⇒ |ξ-x| < δ ⇒ |(Φ(x+h)-Φ(x))/h - f(x)| < ε。
```

## 7. 混合偏导数交换的 ε-δ 证明

**定理：** 若 \(f_{xy}\) 和 \(f_{yx}\) 连续，则 \(f_{xy}=f_{yx}\)。

**证明框架：**
```
令 Δ = f(x₀+h,y₀+k) - f(x₀+h,y₀) - f(x₀,y₀+k) + f(x₀,y₀)
固定 h,k。
令 g(y) = f(x₀+h,y) - f(x₀,y)，则 Δ = g(y₀+k) - g(y₀)
由中值定理：Δ = g'(η)k = [f_y(x₀+h,η) - f_y(x₀,η)]k
再由中值定理：Δ = f_{xy}(ξ,η)hk
交换顺序同理：Δ = f_{yx}(ξ',η')hk
h,k→0 时由连续性得 f_{xy}(x₀,y₀) = f_{yx}(x₀,y₀)。
```

## 8. 级数比较判别法的 ε-N 证明

**定理：** \(0 \le a_n \le b_n\)，\(\sum b_n\) 收敛 ⇒ \(\sum a_n\) 收敛。

**证明框架：**
```
设 B_n = b₁+⋯+b_n → S。
由 ε-N 定义，对任意 ε>0，存在 N 使 m>n>N 时 |B_m - B_n| < ε。
由于 a_n ≤ b_n，A_m - A_n = a_{n+1}+⋯+a_m ≤ b_{n+1}+⋯+b_m < ε。
故 {A_n} 是柯西列，∑ a_n 收敛。
```
